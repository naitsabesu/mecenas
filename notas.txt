Instalación de Redis en Windows
Instalación de Redis en Windows para la clase de hoy martes tenemos que tener instalado redis, y como el punto mas devil de muchos desarrolladores bajo windows es la consola les pongo una guia simple y rapida para instalarlo.
1.- descargar redis de este enlace de github (https://github.com/rgl/redis/downloads)
que a direferencia de los usuarios de linux que tienen que clonar el repositorio y compilarlo, para los de windows es un archivo clasico .exe
2.- instalarlo
3.- para comprobar que esta bien instalado y funcionando hay que ir a la carpeta donde se instalo el programa y ejecutar el archivo "redis-server" esto es para levantar el servidor de redis y poder trabajar con su consola de comandos.
4.- luego abrir el archivo "redis-cli" que es la consola de redis y ponerse a trabajar.
5.- para ir practicando y verificar que todo esta correcto pueden seguir esta guia de redis, tambien funciona online pero les puede ayudar a ir probandola consola antes de la clase link
aqui (http://try.redis.io/)
