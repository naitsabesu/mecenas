<h3>Mecenas</h3>

Proyecto de prueba de nodejs / mongodb / redis y angularjs

<strong>Instalacion</strong>

Simplemente ejecutar dentro de la carpeta "server" del proyecto <br>
<code>npm install</code>

<strong>Requisitos</strong>

<ul>
  <li>Nodejs</li>
  <li>Mongodb</li>
  <li>Redis server</li>
</ul>
